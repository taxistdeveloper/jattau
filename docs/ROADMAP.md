## Фаза 1: MVP — Чтение (Недели 3-6)

- [ ] API: суры, аяты, quran_text
- [ ] Flutter: экраны сур и аятов
- [ ] Запись аудио на устройстве
- [ ] Загрузка аудио на сервер
- [ ] Интеграция Whisper STT
