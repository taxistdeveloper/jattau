# UI/UX Design — Jattau

## Design System

### Цветовая палитра (Material 3)

| Token | Light | Dark | Назначение |
|-------|-------|------|------------|
| primary | `#1B6B4A` | `#4CAF7A` | Islamic green |
| secondary | `#C9A227` | `#E8C547` | Gold accent |
| surface | `#FAFAF8` | `#1A1A1A` | Background |
| on-surface | `#1C1B1F` | `#E6E1E5` | Text |
| error | `#BA1A1A` | `#FFB4AB` | Errors |
| success | `#2E7D32` | `#81C784` | Passed recitation |

### Типографика

| Стиль | Шрифт | Размер | Использование |
|-------|-------|--------|---------------|
| Arabic Quran | Amiri / Scheherazade | 28-36sp | Текст аятов |
| Arabic UI | Noto Naskh Arabic | 16-20sp | UI элементы |
| Headline | Inter Bold | 24sp | Заголовки |
| Body | Inter Regular | 16sp | Основной текст |
| Caption | Inter Medium | 12sp | Подписи |

### Spacing

- Base unit: 4dp
- Screen padding: 16dp
- Card padding: 16dp
- Section gap: 24dp

## UX Principles

1. **Progressive disclosure** — показывать только нужное на каждом шаге
2. **Immediate feedback** — анимация записи, прогресс загрузки
3. **Forgiving** — 5 попыток, подсказки, эталонное аудио
4. **Motivating** — XP, streak, achievements после каждого успеха
5. **Accessible** — крупный арабский шрифт, high contrast, screen reader
6. **RTL support** — арабский текст RTL, UI LTR

## Анимации

| Элемент | Анимация | Duration |
|---------|----------|----------|
| Mic button | Pulse при записи | loop |
| Success | Confetti + scale | 1.5s |
| Error shake | Horizontal shake | 300ms |
| Page transition | Shared axis | 300ms |
| XP gain | Float up + fade | 1s |
| Streak fire | Flame particle | loop |
